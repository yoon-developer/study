COUNT
==========

🔝 [Intermediate Main](./0.%20Intermediate.md)

🔙 [Aggregate Functions](./1.%20AggregateFunctions.md)

***
> 
- count(data): data 값이 NULL 을 제외한 개수 출력

```sql
SELECT
  COUNT(*)
FROM
  tutorial.aapl_historical_stock_price

|-------| 
| count | 
|-------| 
| 3555  | 
|-------| 
```

```sql
SELECT
  COUNT(high)
FROM
  tutorial.aapl_historical_stock_price

|-------| 
| count | 
|-------| 
| 3531  | 
|-------| 
```
